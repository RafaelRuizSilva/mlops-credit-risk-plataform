host de desenvolvimento
docker compose — rede interna (dns: api, mlflow, postgres…)
caminho quente — trabalha a cada clique do usuário antes — prepara o modelo (treino/startup) depois — observa o que aconteceu
No seu clique (milissegundos) Só três peças trabalham: o porteiro entrega e repassa, o cérebro decide com o modelo já na memória, o cartório registra antes de responder. frontend · api · postgres
Antes (para o clique ser possível) A fábrica treina e decide quem é o campeão; o catálogo sabe qual versão vale; o depósito guarda os bytes que a API carrega no startup. dataset · pipelines · mlflow · minio
Depois (observando) O contador mede saúde da API; a TV desenha os gráficos; o fiscal compara quem está chegando com quem o modelo estudou (drift). prometheus · grafana · evidently
